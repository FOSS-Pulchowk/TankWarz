1. First Create a Server 
2. The Server port must be set to 2000
3. Enter the no of players you want to host the game for
4. Open the game again if you want to play from the same computer.
   In that case the port no cannot be 2000.
   Enter the server ip as 127.0.0.1(localhost) or ****.****.****.**** (local ip address)
5. Wait for other players to join the game. The game is joined from other computers by entering the server IP address
   and any port no.
6. The game will start when all the players have joined the server.

8. Use arrow keys to move the tank and mouse to aim.
